# Magyar ékezetek a PDF-ben

A PDF export Noto Sans betűtípust használ a helyes magyar ékezetek megjelenítéséhez.

**Automatikus:** A betűtípusok GitHub-ról töltődnek. Ha ez nem működik (pl. hálózati probléma), letöltheted és ide másolhatod:

1. Nyisd meg: https://github.com/googlefonts/noto-fonts/tree/main/hinted/ttf/NotoSans
2. Töltsd le: `NotoSans-Regular.ttf`, `NotoSans-Bold.ttf`, `NotoSans-Italic.ttf`
3. Másold a fájlokat ebbe a `public/fonts/` mappába
